# Welcome to CANTEEN-HCMUS

## I. Set up

### 1. Install local module using npm

-   Frontend

```
$ cd frontend
$ npm install
```

### 2. Run this project:

-   Backend

```
$ cd backend
$ docker compose up -d
```

-   Frontend

```
$ cd frontend
$ npm run dev
```
